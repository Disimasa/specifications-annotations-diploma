## Параметры обучения

При transfer learning рекомендуется:

1. **Меньший learning rate** (1e-5 вместо 2e-5)
   - Модель уже обучена, нужен аккуратный fine-tuning
   - Избегаем "забывания" предыдущих знаний

2. **Меньше warmup steps** (500 вместо 1000)
   - Модель уже в хорошем состоянии
   - Не нужно долго разогреваться

3. **Меньше эпох** (2-3 вместо 3-5)
   - Модель быстрее адаптируется
   - Меньше риск переобучения

## Сравнение подходов

| Подход | Время обучения | Качество | Рекомендация |
|--------|---------------|----------|--------------|
| **С нуля на STS** | ~2-3 часа | Хорошее | Если нет IR модели |
| **IR → STS (transfer)** | ~1-2 часа | **Лучшее** | ✅ **Рекомендуется** |
| **Только IR** | - | Плохое для STS | ❌ Не подходит |
