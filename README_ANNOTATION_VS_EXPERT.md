1. **Литейные сплавы** (0.879)
2. **Фундаментальные исследовательские компетенции** (0.861)
3. **Проектирование производства** (0.855)

### Экспертная оценка (топ-3):
1. **Плавка и заливка** (очень высокая релевантность)
2. **Проектирование производства** (высокая релевантность)
3. **Фундаментальные исследовательские компетенции** (средняя релевантность)

### Анализ соответствия:
- ✅ **Проектирование производства** - правильно определено (3-е место в аннотации, 2-е в экспертной оценке)
- ✅ **Фундаментальные исследовательские компетенции** - правильно определено (2-е место в аннотации, 3-е в экспертной оценке)

## Выводы и рекомендации

### Сильные стороны:
1. ✅ **"Проектирование производства"** - очень хорошо определяется для файлов про автоматизацию и мониторинг
2. ✅ **"Фундаментальные исследовательские компетенции"** - хорошо определяется для файлов с математическими моделями
3. ✅ **"Литейные сплавы"** - иногда правильно определяется

### Основные проблемы:
1. ❌ **Путаница между семантически близкими компетенциями:**
   - "Литейные сплавы" vs "Плавка и заливка" (оба про металл)
   - "Формовочные материалы" vs "Отливки" (оба про материалы/изделия)

2. ❌ **Отсутствие ключевых компетенций:**
   - "Плавка и заливка" для аргона
   - "Отливки" для мелющих шаров
   - "Специальные способы литья" для аддитивного производства

3. ❌ **Добавление нерелевантных компетенций:**
   - "Формовочные материалы" для прокатки
   - "Другие виды литья" для прокатки и IT

### Рекомендации по улучшению:
1. **Улучшить описания компетенций в онтологии:**
   - Добавить ключевые слова и примеры использования
   - Четко разграничить семантически близкие компетенции
   - Добавить в "Специальные способы литья" упоминание аддитивного производства

2. **Использовать cross-encoder для re-ranking:**
   - Может помочь лучше различать семантически близкие компетенции
   - Улучшит порядок компетенций

3. **Добавить предметные правила/фильтры:**
   - Исключать "Формовочные материалы" для файлов про прокатку
   - Исключать "Другие виды литья" для файлов про IT

4. **Расширить контекстное окно:**
   - Может помочь лучше понимать контекст и различать компетенции
